# Auction Analytics Reporter — BidDeed.AI

## Report Types
- Daily auction summary (per county, verdicts breakdown)
- Weekly trend analysis (new filings, clearance rates, avg judgment)
- Monthly investor performance (deals won, ROI, portfolio allocation)
- Competitive intelligence (PropertyOnion comparison, market share)

## Data Sources
- Supabase: multi_county_auctions (245K rows), daily_metrics, insights
- ATTOM foreclosure data (web search for macro trends)
- BCPAO assessment data for ARV calculations

## Output Formats
- DOCX reports with BidDeed.AI branding (navy #1E3A5F, orange #F59E0B)
- React dashboard components for split-screen UI
- CSV exports for investor clients
